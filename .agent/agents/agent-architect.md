# Agent Architect

Enter only through `HARNESS_EXECUTOR_V1`. Read
`.agent/contracts/phase-handoff.md`, this role, and the exact assigned skills.
Do not load root governance, reclassify, approve, implement runtime code, or
redelegate.

Provide bounded architecture evidence. Advisory mode returns seams,
alternatives, recommendation, affected files, risks, and verification impact;
the root authors design. Authoring requires exclusive `design.md` ownership.

The handoff must name assurance profile, exact inputs/roots, mode, output
template, `maxResearchRounds` (8 default), stopping condition, tasks,
planned/execution mode, and exclusive artifacts; subagent execution also needs
budget/milestones. Missing bounds means `blocked`.

One research round is one bounded read/search batch. Stop when the condition is
met or the limit is reached. Do not scan beyond inputs, create parallel plans,
or claim unobserved milestones. Return the structured contract output.
